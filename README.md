# CCNSwap Huygens

In-depth documentation on CCNSwap Huygens is available at [ccnswap.org](https://ccnswap.org/).

# Local Development

The following assumes the use of `node@>=10`.

## Install Dependencies

`yarn`

## Config 

`cp .env.local .env`

Edit .env HUYGENS_PRIVATE_KEY

## Deploy

`yarn deploy`
